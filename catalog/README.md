# NTE200 — Интерактивный каталог запчастей

Кликабельная веб-версия книги запчастей карьерного самосвала **NTE200**
(Inner Mongolia North Hauler, NHL): чертежи узлов, таблицы деталей и
корзина для формирования заявки на заказ.

## Как открыть

Каталог полностью статический и работает без сервера — просто откройте
`catalog/index.html` в браузере (двойной клик).

Для локального веб-сервера (необязательно):

```bash
cd catalog
python3 -m http.server 8000   # затем http://localhost:8000
```

## Возможности

- **Навигация по системам** — разделы сгруппированы по главам книги
  (структура, электрика, гидравлика, ходовая, кабина и т. д.).
- **Чертежи** — изображение каждого узла; клик открывает просмотр с
  масштабированием (колесо мыши / щипок) и перетаскиванием.
- **Таблицы деталей** — номер позиции, номер детали, наименование (кит./англ.),
  количество; отступами показана иерархия сборок (подкомпоненты).
- **Поиск** — мгновенный поиск по номеру детали или названию сразу по всему
  каталогу (2 300+ позиций).
- **Корзина заказа** — добавление деталей кнопкой ＋, редактирование
  количества, поле серийного номера машины, выгрузка заявки в **CSV** или на
  **печать**. Содержимое корзины сохраняется в браузере (localStorage).

## Структура

```
catalog/
  index.html         # разметка приложения
  styles.css         # оформление
  app.js             # логика (без внешних зависимостей)
  data/parts.js      # данные каталога: window.CATALOG = {...}
  drawings/*.jpg      # чертежи (по одному на страницу-иллюстрацию книги)
```

## Как пересобрать данные из PDF

Данные и изображения извлекаются из исходного PDF скриптом
[`tools/extract_catalog.py`](../tools/extract_catalog.py). PDF хранится в
репозитории в виде многотомного архива (`NTE200 PART номера Polyus.zip.001…004`),
и скрипт при необходимости сам собирает его обратно.

```bash
pip install pymupdf
python3 tools/extract_catalog.py            # соберёт PDF из zip-томов
# или указать путь к PDF напрямую:
python3 tools/extract_catalog.py path/to/catalog.pdf
```

Скрипт заново создаёт `catalog/data/parts.js` и `catalog/drawings/`.
