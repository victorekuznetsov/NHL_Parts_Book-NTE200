# NTE200 — Интерактивный каталог запчастей

Кликабельная веб-версия книги запчастей карьерного самосвала **NTE200**
(Inner Mongolia North Hauler, NHL): чертежи узлов, таблицы деталей и
корзина для формирования заявки на заказ.

## Как открыть

Каталог полностью статический и работает без сервера — просто откройте
`catalog/index.html` в браузере (двойной клик).

Для локального веб-сервера (необязательно):

```bash
cd catalog
python3 -m http.server 8000   # затем http://localhost:8000
```

## Возможности

- **Навигация по системам** — разделы сгруппированы по главам книги
  (структура, электрика, гидравлика, ходовая, кабина и т. д.).
- **Чертёж ↔ список позиций** — каждый чертёж показан рядом со своим списком
  позиций. Если в разделе несколько отдельных рисунков (в книге они идут
  парами «рисунок → таблица»), они разбиты на «Рисунок 1 / N · позиции …», и
  под каждым — только относящиеся к нему позиции. Если один узел показан
  несколькими листами с общим списком, листы собраны в карусель (‹ ›).
- **Чертежи** — клик открывает просмотр с масштабированием
  (колесо мыши / щипок) и перетаскиванием.
- **Таблицы деталей** — номер позиции, номер детали, наименование (кит./англ.),
  количество на схеме; отступами показана иерархия сборок. Позиции, у которых
  в книге нет отдельного каталожного номера (компоненты ремкомплектов),
  показаны с «—», чтобы список совпадал с номерами на чертеже.
- **Выбор требуемого количества** — в каждой строке рядом с количеством на
  схеме есть поле «Нужно»; кнопка ＋ добавляет в заказ именно это число.
  Количество на схеме может быть числом или «AR» (As Required / по потребности).
- **Светлое оформление** и строка контактов вверху: по вопросам работы каталога,
  ошибок и замечаний — Кузнецов В.Е., KuznetsovVE@industrservice.ru.
- **Поиск** — мгновенный поиск по номеру детали или названию сразу по всему
  каталогу (2 300+ позиций).
- **Корзина заказа** — добавление деталей кнопкой ＋, редактирование
  количества, поле серийного номера машины, выгрузка заявки в **CSV** или на
  **печать**. Содержимое корзины сохраняется в браузере (localStorage).
- **Экспорт всех каталожных номеров** — кнопка на стартовом экране и в корзине
  выгружает CSV со всеми уникальными каталожными номерами (с указанием
  источника NTE200 / GE и разделов). Готовый файл также лежит в
  `data/all_part_numbers.csv`.

## Содержимое

Каталог включает книгу запчастей NTE200 и **привод GE** (глава 600 —
驱动系统 / DRIVING SYSTEM):轮马达/WHEEL MOTOR, 主发电机/MAIN ALTERNATOR,
电阻柜/RETARDER, 电控柜/CONTROL GROUP, 接触器/CONTACTOR.

```
catalog/

```

## Как пересобрать данные из PDF

Данные и изображения извлекаются из исходного PDF скриптом
[`tools/extract_catalog.py`](../tools/extract_catalog.py). PDF хранится в
репозитории в виде многотомного архива (`NTE200 PART номера Polyus.zip.001…004`),
и скрипт при необходимости сам собирает его обратно.

```bash
pip install pymupdf
python3 tools/extract_catalog.py            # соберёт PDF из zip-томов
# или указать путь к PDF напрямую:
python3 tools/extract_catalog.py path/to/catalog.pdf
```

Скрипт заново создаёт `catalog/data/parts.js` и `catalog/drawings/`.

### Привод GE (глава 600)

Данные и чертежи привода GE извлекаются из Word-документа
`NTE200 GE备件手册20230805.doc` скриптом
[`tools/extract_ge.py`](../tools/extract_ge.py). Документ — бинарный `.doc`
(OLE2); текст читается напрямую из piece table, чертежи вынимаются как PNG из
потока `Data` и раскладываются по разделам по числу якорей-картинок. Скрипт
дописывает главу 600 в `parts.js`, сохраняет `catalog/drawings/600-*.png` и
пересобирает `catalog/data/all_part_numbers.csv`.

```bash
pip install olefile
python3 tools/extract_ge.py "NTE200 GE备件手册20230805.doc"
```
